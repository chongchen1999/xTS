# Preface

> 本书面向所有程序员，通过类型系统让你的程序更安全、更易维护。

## 本书结构

- 前几章：编译器、类型检查器、类型系统（理论）
- 中间章节：高级类型特性、错误处理、异步编程
- 后几章：框架集成、JS 迁移、生产部署（实践）

## 代码风格建议

- 使用最新 JavaScript 语法（esnext）
- 用展开运算符 `...` 保持数据不可变
- 尽量让 TypeScript 自动推断类型，避免过度显式标注
- 保持代码通用和可复用，善用多态

```typescript
// immutable update
const o = { a: 1, b: 2 }
const p = { ...o, b: 3 } // { a: 1, b: 3 }
```
